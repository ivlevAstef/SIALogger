# SIALogger
library for simplify log and assertion

## Features

1. Five log levels:
  * Fatal
  * Error
  * Warning
  * Info
  * Trace
2. Assertion logs - in debug mode abort program, but in release mode only write message.
3. Log by condition (LogIf)
4. Settings
